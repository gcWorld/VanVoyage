# App Icons

Place launcher icons in the following directories:
- mipmap-mdpi: 48x48
- mipmap-hdpi: 72x72
- mipmap-xhdpi: 96x96
- mipmap-xxhdpi: 144x144
- mipmap-xxxhdpi: 192x192

These icons should be named `ic_launcher.png`

You can generate icons using tools like:
- Android Studio's Image Asset Studio
- https://romannurik.github.io/AndroidAssetStudio/
- https://appicon.co/

For now, the app will use the default Flutter icon.
